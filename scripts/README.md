# LUAD ML Pipeline
## Machine Learning Pipeline for LUAD Candidate Gene Prioritization and Co-expression Network Visualization

---

## Project Structure

```
LUAD_ML_Pipeline/
├── config.py                          # Step 0 - Central configuration
├── step1_data_loading.py              # Step 1 - Load raw data files
├── step2_preprocessing.py             # Step 2 - QC & log2 normalization
├── step3_harmonization.py             # Step 3 - Gene ID harmonization
├── step4_differential_expression.py   # Step 4 - Welch t-test + BH FDR
├── step5_expression_features.py       # Step 5 - Build expression features
├── step6_coexpression_network.py      # Step 6 - Build co-expression network
├── step7_network_features.py          # Step 7 - Extract network topology features
├── step8_feature_integration.py       # Step 8 - Merge all features
├── step9_label_construction.py        # Step 9 - Assign binary labels (CGC)
├── step10_train_val_split.py          # Step 10 - Stratified train/val split
├── step11_model_training.py           # Step 11 - Train 5 ML models
├── step12_model_evaluation.py         # Step 12 - Evaluate on held-out set
├── step13_feature_importance.py       # Step 13 - MDI + permutation importance
├── step14_gene_ranking.py             # Step 14 - Rank all genes genome-wide
├── step15_network_annotation.py       # Step 15 - Annotate network with ML results
├── step16_network_export.py           # Step 16 - Export network (GraphML, GML, TSV, JSON)
├── step17_interactive_visualization.py# Step 17 - Plotly interactive HTML dashboard
├── step18_final_report.py             # Step 18 - Final reports and figures
├── run_pipeline.py                    # Master runner (auto-generated by Step 18)
└── requirements.txt
```

## Usage

### Run all 18 steps at once:
```bash
cd LUAD_ML_Pipeline
python run_pipeline.py
```

### Run individual steps:
```bash
python config.py                           # Step 0 - Validate config
python step1_data_loading.py               # Step 1
python step2_preprocessing.py              # Step 2
python step3_harmonization.py              # Step 3
python step4_differential_expression.py    # Step 4
python step5_expression_features.py        # Step 5
python step6_coexpression_network.py       # Step 6  (slow: ~60-90s)
python step7_network_features.py           # Step 7  (slow: ~90s)
python step8_feature_integration.py        # Step 8
python step9_label_construction.py         # Step 9
python step10_train_val_split.py           # Step 10
python step11_model_training.py            # Step 11 (slow: ~5-10 min)
python step12_model_evaluation.py          # Step 12
python step13_feature_importance.py        # Step 13 (slow: ~2-3 min)
python step14_gene_ranking.py              # Step 14
python step15_network_annotation.py        # Step 15
python step16_network_export.py            # Step 16
python step17_interactive_visualization.py # Step 17 (requires: pip install plotly)
python step18_final_report.py              # Step 18
```

## ML Models Trained

| Model | Features |
|-------|----------|
| Random Forest (best) | Unscaled |
| Gradient Boosting | Unscaled |
| Extra Trees | Unscaled |
| Logistic Regression | RobustScaler |
| SVM (RBF) | RobustScaler |

All models use `class_weight='balanced'` to handle ~17:1 class imbalance.
Primary metric: AUROC. Secondary: AUPRC, MCC.

---

## Configuration

All parameters are in `config.py`:
- `COEXPR_CORRELATION_CUTOFF = 0.70` - raise to 0.80 for faster Step 6
- `CV_FOLDS = 5` - number of cross-validation folds
- `DE_LOG2FC_THRESHOLD = 1.0` - absolute log2FC threshold
- `DE_PVALUE_THRESHOLD = 0.05` - BH-adjusted p-value threshold
